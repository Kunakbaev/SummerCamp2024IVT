\doxysubsection{Detailed Description}
main struct 

Quadratic equation looks like this\+: a $\ast$ x $^\wedge$ 2 + b $\ast$ x + c.

Class can find solutions of quadratic equation in rationals, also output\+Precisoin can be state (10 by default), helps outputing floating number.
